MOVOTO
Web . iOS . Android

Movoto is a full-service real estate brokerage that provides customers with search tools and information on top agents. One of my first projects for Movoto was to design the mobile experiences.

The Challenge

My main task for this project was to enhance the mobile design of Movoto's search and discovery products. As the UI/UX Designer, I guided the UI Toolkit and provided direction for both mobile web and native mobile products. The ultimate goal was to create a visually appealing and functional mobile experience for users.

The UI Tool Kit

As we were making rapid progress, I generated a Photoshop and redline file containing the specifications for the front-end developers. This way, they could access all the necessary design assets in one location, ensuring a consistent appearance and user experience.
