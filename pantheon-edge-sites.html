PANTHEON
Edge Sites

Pantheon is a platform for managing Drupal and WordPress websites. It helps companies, agencies, and developers handle multiple sites from one dashboard. It optimizes work efficiency and offers version control, staging environments, backups, and workflow.

The Challenge

Pantheon has a large user base on Drupal and WordPress sites. We needed to develop a modern offering for building headless CMS websites and a framework to support WebOps teams in designing and creating content for future products.

The Process

We needed to quickly provide additional data to pass the business case review and construct the product.

Design Sprint

To create a prototype of a developer-focused static site solution with code that can be user-tested at the end of the sprint.

Engineering Goal

It is essential to have a perspective when it comes to coding and technology.

Design Goal

Our hypothesis will be validated through a UX prototype encompassing the entire Pantheon experience, both on and off.

The Results

  • The prototype took less than 5 minutes for most users, with 100% completing all tasks confidently.
  • All participants showed interest in the product offering.

The Product

Connect with GitHub

If you're moving your website to Pantheon, connecting with GitHub can help you integrate your current repositories seamlessly.

Site Overview

On the site overview page, you can check the status of your live build and view a list of all available branches along with their respective details.

Build Details

On the build details page, you can view the site's most recent status and metrics.

Lighthouse Scores

A Google Lighthouse score is a rating that Google gives to a website based on performance, accessibility, SEO, and best practices.

Real-time Build Logs

A build log is a structured list of events during a build. Live logging allows you to view the progress of a build in real time as each step runs. Logging can make it easier for software developers to monitor, troubleshoot, and debug.
